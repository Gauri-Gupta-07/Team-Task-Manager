# Team-Task-Manager

# 📝 Task Manager Project

## 📌 Overview
This is a Task Manager Web Application that allows users to create, update, delete, and manage daily tasks efficiently.

---

## 🎯 Features
- Add new tasks
- Update existing tasks
- Delete tasks
- View all tasks
- User-friendly interface

---

## 🛠️ Technologies Used
- Java (Backend)
- HTML, CSS, JavaScript (Frontend)
- MySQL (Database)
- JDBC / Servlet
