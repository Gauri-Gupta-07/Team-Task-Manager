# Team-Task-Manager

# 📝 Task Manager Project (Spring Boot)

## 📌 Overview
This is a Task Manager web application built using Spring Boot. It is designed to help users manage their daily tasks efficiently by providing basic task operations.

---

## 🚀 Features
- Create new tasks
- View all tasks
- Update existing tasks
- Delete tasks
- Backend powered by Spring Boot

---

## 🛠️ Technologies Used
- Java
- Spring Boot
- Maven
- REST APIs
- PostgreSQL (can be configured)
